# Papers — MAM5020F 2026: Generative AI for Research

This folder tracks the academic papers cited across the course. **PDFs are no longer stored or redistributed here** — instead, the papers section of the course site links to each paper's canonical source (arXiv abstract, journal DOI, or publisher page).

- **Published list (with links):** [`docs/papers/`](../docs/papers/) — one card per paper, grouped by week, generated by `build_papers_site.py`.
- **`Week-N/MANIFEST.md`** — per-week reference notes (title, authors, year, sub-lesson mapping, DOI). Retained as a working record; the authoritative, link-bearing version is the generated site above.

## Scope

- **In scope:** academic papers — arXiv preprints, journal articles with DOIs, conference papers (NeurIPS, FAccT, CHI, etc.).
- **Out of scope:** blog posts, tool documentation, books, news articles, software repositories. Those remain linked in-line on the lesson pages.

## Editing the published list

The paper data lives in the `WEEKS` dict in `build_papers_site.py`. Edit it there and re-run:

```bash
python3 build_papers_site.py
```

Each entry carries a `src_url` (canonical link) and `src_label` (e.g. `arXiv:2304.03271` or `DOI:…`). Paywalled items go in a week's `paywalled` list with the DOI for UCT-library access.
